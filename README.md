# Prediction Live Dashboard

Analytics dashboard for the Prediction Live smart contract deployed on Chiliz Testnet.

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Build for Production

```bash
npm run build
npm start
```

## Deploy on Vercel

1. Push your code to GitHub
2. Import your repository on [Vercel](https://vercel.com)
3. Vercel will automatically detect Next.js and configure the build settings
4. Deploy!

The easiest way to deploy your Next.js app is to use the [Vercel Platform](https://vercel.com/new).

## Features

- Real-time analytics from smart contract transactions
- Historical trends with interactive charts
- Dark mode UI
- Responsive design

## Tech Stack

- Next.js 14
- React 18
- Chart.js with react-chartjs-2
- TypeScript
